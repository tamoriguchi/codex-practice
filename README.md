# LINE返信テンプレ管理ツール MVP

小規模店舗・個人事業者が、LINEやDMで繰り返し使う返信文を保存し、検索してワンクリックでコピーできる小型ツールの試作です。

## 選定理由

30個の候補を「支払い意欲」「利用頻度」「実装容易性」「差別化しやすさ」「運用リスク」の5軸で比較し、最も成功確率が高い候補として **LINE返信テンプレ管理ツール** を選びました。詳細は [`docs/product-plan.md`](docs/product-plan.md) を参照してください。

## 現在のMVP範囲

- 返信テンプレートの登録
- カテゴリ管理
- キーワード検索
- ワンクリックコピー
- ブラウザの `localStorage` への保存
- サンプルテンプレートの同梱

## 使い方

依存関係はありません。任意の静的サーバーで起動できます。

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開きます。

## 想定ユーザー

- 美容室、整体院、ネイルサロン、個人教室などの小規模店舗
- 予約・料金・キャンセル・道案内など、同じ返信を何度も送る事業者
- LINE公式アカウントやInstagram DMを手作業で運用している人
