# 小型ツール候補の比較と実装準備

## 結論

最初に作るべき小型ツールは **LINE返信テンプレ管理ツール** です。

理由は、毎日使われる可能性が高く、導入説明が簡単で、1〜3日で価値が伝わるMVPを作りやすいからです。特に美容室、整体院、ネイルサロン、個人教室などは予約・料金・キャンセル・道案内などの定型返信が多く、月額980円前後の小額課金と相性が良いと判断しました。

## 比較軸

| 評価軸 | 内容 | 重み |
| --- | --- | ---: |
| 支払い意欲 | お金を払ってでも解決したい痛みか | 30% |
| 利用頻度 | 毎日または毎週使われるか | 25% |
| 実装容易性 | 1〜3日でMVPを作れるか | 20% |
| 差別化しやすさ | 業種特化やテンプレートで売り分けられるか | 15% |
| 運用リスク | 法務・金融・外部API依存などの重さが低いか | 10% |

## なぜLINE返信テンプレ管理を選ぶか

### 1. 利用シーンが明確

小規模店舗では、以下のような返信が日常的に発生します。

- 予約可能日時の案内
- 料金表の案内
- キャンセルポリシーの説明
- 店舗住所や駐車場の案内
- 施術前の注意事項
- 来店後のお礼や口コミ依頼

「返信文を探してコピーする」という単純な作業でも、毎日発生すれば時間削減効果が見えやすくなります。

### 2. MVPが小さい

初期版は次の機能だけで成立します。

- テンプレート登録
- カテゴリ分類
- 検索
- コピー
- 履歴保存

外部API連携や複雑な権限管理なしで、ユーザーに価値を見せられます。

### 3. 売り方を業種特化にできる

汎用の「テンプレート管理」ではなく、次のように売ることで刺さりやすくなります。

- 美容室向けLINE返信テンプレ集
- 整体院向け予約・キャンセル返信管理
- ネイルサロン向けDM返信定型文ツール
- 個人教室向け体験レッスン返信管理

同じプロダクトでも、初期テンプレートとLPの文言を変えるだけで複数の市場を試せます。

## MVP仕様

### 画面

1. テンプレート一覧
2. テンプレート登録・編集フォーム
3. カテゴリフィルタ
4. 検索欄
5. コピー完了通知

### データ項目

| 項目 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| id | string | yes | テンプレートID |
| title | string | yes | テンプレート名 |
| category | string | yes | 予約、料金、キャンセルなど |
| body | text | yes | コピーする本文 |
| updatedAt | string | yes | 更新日時 |

### 無料・有料の切り分け案

| プラン | 月額 | 制限 |
| --- | ---: | --- |
| Free | 0円 | テンプレート5件まで |
| Lite | 680円 | テンプレート50件、カテゴリ10個 |
| Pro | 1,480円 | テンプレート無制限、スタッフ共有、業種別テンプレート |

## 初期ターゲット

最初の検証対象は **美容室・整体院・ネイルサロン** に絞ります。

理由は、予約連絡・キャンセル連絡・事前案内・来店後フォローが多く、LINE公式アカウントやInstagram DMで定型返信を使う頻度が高いからです。

## 1〜3日の実装計画

### Day 1

- 静的プロトタイプ作成
- テンプレート登録・一覧・検索・コピー
- localStorage保存

### Day 2

- Supabaseなどでログインとクラウド保存
- テンプレート編集・削除
- 業種別サンプルテンプレート追加

### Day 3

- 課金前LP作成
- Free/Lite/Proの制限ロジック
- Stripe Checkout導入準備
- 事前登録フォーム設置

## 検証指標

- LP訪問者の事前登録率: 5%以上
- デモ利用者のテンプレート登録率: 30%以上
- コピー機能の利用回数: 1ユーザーあたり週5回以上
- 有料化アンケートで月額680円以上の支払い意思: 20%以上

## リスクと対策

| リスク | 対策 |
| --- | --- |
| メモアプリで十分と思われる | 業種別テンプレート、カテゴリ、検索、コピー導線で差別化する |
| LINE公式と直接連携を期待される | 初期版は「コピーして貼る」ツールとして明確に位置付ける |
| 継続課金の理由が弱い | テンプレート数制限、スタッフ共有、業種別文面パックを有料価値にする |
| 汎用すぎて刺さらない | 最初は美容室・整体院・ネイルサロンに絞ってLPを作る |
